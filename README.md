## Description

This project is a journey from a Class to a Game. first commits will set the basis, and progressivly transform, chapter after chapter, the initial class into a playable game.

## build

```bash
mvn clean install
```

## test

```bash
mvn test
```

## run

```bash
mvn exec:java
```
